ThekSelect

A headless, framework-agnostic, and accessible select library.

Theme Preview

Change the theme above to see how the components adapt.

Basic Usage

Simple single-select initialized from an existing <select> element.

Events will appear here...
ThekSelect.init('#basic-select', {
    placeholder: 'Pick a fruit'
});

Multi-select & Tagging

Support for multiple selections with reorderable tags via Drag-and-Drop.

Events will appear here...
ThekSelect.init('#multi-select', {
    placeholder: 'Select frameworks',
    multiple: true
});

Custom Tag Creation

Allow users to create new options on the fly by typing.

Events will appear here...
ThekSelect.init('#create-container', {
    canCreate: true,
    multiple: true,
    options: [
        { value: 'js', label: 'JavaScript' },
        { value: 'ts', label: 'TypeScript' }
    ],
    createText: "Add new skill: '{%t}'"
});

Remote Data Loading

Fetch options from an API with built-in debouncing.

Events will appear here...
ThekSelect.init('#remote-container', {
    searchable: true,
    placeholder: 'Search GitHub Users...',
    loadOptions: async (query) => {
        const res = await fetch(`https://api.github.com/search/users?q=${query}`);
        const data = await res.json();
        return data.items.map(u => ({
            value: u.login,
            label: u.login
        }));
    }
});
